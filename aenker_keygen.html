aenker keygen

generate a new key

Synopsis

Generate and save a new random Curve25519 keypair.

aenker keygen [flags]

Examples

  aenker kg -p publickey -o secretkey

Options

  -f, --file string      save key to this file (default "/home/ansemjo/.local/share/aenker/aenkerkey")
  -c, --comment string   add comment to keyfile
  -h, --help             help for keygen
