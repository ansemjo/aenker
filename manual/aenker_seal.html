aenker seal

encrypt and protect a file

Synopsis

Encrypt a file for a recipient's public key and output authenticated ciphertext.

aenker seal [flags]

Examples

  tar -cz * | aenker seal -p $PUBLICKEY > archive.tar.gz.ae

Options

  -p, --peer string     receiver's public key
  -i, --input string    input file, plaintext (default: stdin)
  -o, --output string   output file, ciphertext (default: stdout)
  -h, --help            help for seal
